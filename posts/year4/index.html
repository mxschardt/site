Лабораторная работа направлена на глубокое изучение архитектуры OAuth 2.0 и OpenID Connect (OIDC) с флоу Authorization Code Flow и Proof Key for Code Exchange (PKCE). Нужно создать как минимум три приложения с использованием Spring Boot для моделирования взаимодействий между клиентом (Client), сервером авторизации (Authorization Server) и сервером ресурсов (Resource Server).
